# Local Business Starter Kit

A modern, reusable Next.js starter kit for local business websites.

Built to create fast, SEO-friendly websites for service-based businesses while keeping the codebase clean, scalable and easy to customize.

---

## ✨ Features

- Modern responsive design
- Variant-based architecture
- Reusable UI components
- SEO optimized
- Contact form with Resend
- Google Maps integration
- Structured Data (JSON-LD)
- Framer Motion animations
- Type-safe content
- Ready for Vercel deployment

---

## 🛠 Tech Stack

- Next.js 16
- TypeScript
- Tailwind CSS 4
- Framer Motion
- Resend
- Lucide Icons
- Vercel

---

## 📂 Project Structure

```text
app/
components/
content/
public/
types/
```

The project separates:

- UI components
- business content
- company data
- reusable variants

making it easy to build websites for different industries.

## 🚀 Getting Started

Install dependencies

```bash
npm install
```

Run development server

```bash
npm run dev
```

Build production version

```bash
npm run build
```

---

## 🎨 Creating a New Website

Creating a new project usually requires changing only:

- company information
- content
- branding
- images

The reusable components remain the same.

---

## 🚀 Deployment

The project is optimized for deployment on Vercel.

```bash
npm run build
```
